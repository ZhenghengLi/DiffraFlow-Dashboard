About

DiffraFlow (Diffraction Image Data Flow) is a distributed software developed for the extremely high volume data acquisition and online analysis of pixel array detectors that are typically used with the X-ray Free Electron Laser (XFEL) facility. This software is designed to have good horizontal scaling capability that can run on tens of computer nodes to handle the data stream of up to 100 GiB/s from pixel array detectors. There are several different components in this software, they are dispatcher, combiner, ingester, monitor, controller and aggregator. And they are responsible for different tasks including data receving, online event-building, calibration, deep event filtering, data saving, quasi-realtime analysis for fast feedback, runtime parameters adjustment and system working status monitoring.
